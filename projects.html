Projects

OBAT.IN

OBAT.IN is a project created for the purpose of the Website Design competition organized by the Informatics Engineering Student Association of Gunadarma University. The core objective of the OBAT.IN project is to serve as a prominent digital health platform in Indonesia, offering a user-friendly interface that is easily comprehensible for the general public. OBAT.IN is built using the Bootstrap 5 framework, supported by HTML and CSS, along with a touch of JavaScript scripting.

SIKAP UMKM

"SIKAP UMKM" stands for "Synergy and Collaboration of Small and Medium Enterprises (SMEs) Developers in Bogor City." It was established for the Bogor City Department of Economy and serves as a tool for mapping SMEs within the city. SIKAP UMKM relies on Bootstrap 5 as its CSS framework, MySQL as its primary database system, and is fortified with security measures such as CSP, CORS, Anti-Clickjacking, XSS protection, and more.

SIMONEV

SIMONEV DOKLING is a Licensed Business Activity Mapping System created for the Bogor City Environmental Agency. Its purpose is to map all registered Small and Medium Enterprises (UMKM) distributed across the Bogor City area. The system is built using the Laravel 9 framework, Leaflet as an interactive map, Bootstrap 5 as the CSS framework, and MySQL as the main database. Additionally, the system incorporates security measures such as CSP, CORS, Anti-Clickjacking, XSS protection, etc ensuring a secure usage experience for clients.
